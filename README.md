Holiday Resorts Website

Course: Application of Information and Communication Technology Lab Instructor: Ms. Ramsha Khalid

Project Description: This project is a static website developed using HTML and CSS. The Holiday Resorts website helps users explore different holiday resorts, view basic details, and make booking inquiries. The focus of the project is simple design, clear navigation, and user-friendly layout.
